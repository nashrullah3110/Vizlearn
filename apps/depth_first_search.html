Algorithm

1. Push start node to stack
2. While stack is not empty:
  a. Pop node from stack
  b. Mark as visited
  c. For each neighbor:
    - If not visited
    - Push to stack

Complexity

Time: O(V + E)
Space: O(V)
V = vertices, E = edges
